## 快速上手

### CDN 引入

<br/>

在页面上引入 `js` 和 `css` 文件即可开始使用：

<br/>

**( 因为尚未发布，这个只是个例子 )**

```html
<!-- import stylesheet -->
<link rel="stylesheet" href="暂未发布" />
<!-- import iView -->
<script src="暂未发布"></script>
```

### NPM 安装

<br/>

推荐使用 `npm` 来安装，享受生态圈和工具带来的便利，更好地和 `webpack` 配合使用

```shell
$ npm install vui-vue --save
```
